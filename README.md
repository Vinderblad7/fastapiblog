# FastAPI Blog API 🚀

Асинхронный бэкенд для платформы блогов, построенный на современных инструментах экосистемы Python. Проект создан в качестве практического освоения архитектуры REST API, работы с реляционными базами данных и обеспечения безопасности данных.

## 🛠 Стек технологий

* **FastAPI** — высокопроизводительный асинхронный веб-фреймворк.
* **SQLAlchemy (2.0+)** — ORM для работы с базой данных в асинхронном режиме.
* **Alembic** — управление миграциями и структурой базы данных.
* **Pydantic (v2)** — валидация входящих и исходящих данных (схемы).
* **SQLite + aiosqlite** — легковесная база данных с асинхронным драйвером.
* **JWT (JSON Web Tokens) & Passlib** — хеширование паролей (bcrypt) и безопасная авторизация пользователей.

## 🏗 Архитектура и Функционал

В проекте реализована модульная структура кода. Каждый модуль (`auth`, `posts`, `tags`, `comments`) изолирован и содержит свои роуты, модели БД и Pydantic-схемы.

### Основные возможности:
1.  **Пользователи и Безопасность (`Auth`)**:
    * Регистрация и аутентификация пользователей.
    * Выдача JWT access-токенов.
    * Защита приватных эндпоинтов с помощью системы зависимостей (`Depends`).
2.  **Управление публикациями (`Posts` & `Tags`)**:
    * Полноценный CRUD для постов.
    * Связи «многие-ко-многим» с тегами (`ManyToMany`).
    * Каскадное удаление данных.
3.  **Система комментирования (`Comments`)**:
    * CRUD для комментариев с привязкой к автору и посту.
    * Динамическое обновление полей (`PATCH`) через рефлексию Python (`setattr`).
    * Пагинация списков (`limit`, `offset`) для оптимизации нагрузки на БД.

## 🚀 Локальный запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/Vinderblad7/fastapiblog.git](https://github.com/Vinderblad7/fastapiblog.git)
   cd fastapiblog/backend```

2. Создайте и активируйте виртуальное окружение
python -m venv venv
# Для Windows:
venv\Scripts\activate

# Для macOS/Linux:
source venv/bin/activate

3. Установите зависимости
pip install -r requirements.txt

4. Примените миграции Alembic для создания базы данных
alembic upgrade head

5. Запустите сервер разработки
uvicorn src.main:app --reload
